# Проект "Новости" (объектная модель)

## Запуск
1. Убедитесь, что установлен `uv` и все зависимости (команда `make install`).
2. Выполните `make project` или `uv run python -m src.main`.

## Ожидаемый вывод
Программа получает тестовые данные от mock-источника, обрабатывает их (нормализует текст, считает слова, отфильтровывает пустые записи) и выводит отчёт с количеством обработанных записей и статистикой.
